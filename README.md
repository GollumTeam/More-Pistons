MorePistons
============

More Pistons Mod for Minecraft


Install forge for compile source

For Install :

<pre>
export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-amd64
ln -s ../../GollumCoreLib/GollumCoreLib-1.8.8 GollumCoreLib
./gradlew setupDecompWorkspace
./gradlew build
./gradlew eclipse
<pre>
